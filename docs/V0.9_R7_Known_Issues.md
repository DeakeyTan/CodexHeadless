## Build Environment

The source workspace is inside iCloud Drive. Xcode 27 code signing can reject XCTest products placed in the workspace `.build` directory because iCloud adds extended metadata. Automated tests use an isolated `/private/tmp` SwiftPM scratch path. This does not change production code or test behavior.

## UAT Policy

The unsigned artifact is Internal UAT Only. Private display and Touch Bar APIs may change with macOS updates. No unresolved display-safety defect may be accepted as a Known Issue.
