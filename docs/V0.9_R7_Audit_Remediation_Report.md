## Architecture Decisions

- The virtual display is accepted only when a single structured ready event identifies an enumerated CodexHeadless managed display and the owned host identity remains stable.
- Orphan discovery is token-level; ownership-sensitive actions require canonical executable, file identity and process start identity.
- Clean Normal is a reusable assessment, not an alias for `RuntimeState.mode == Normal`.
- Confirmation suppression is local transient intent and does not replace durable Core state.
- The test-only process helper is a target dependency, not a shipped product; release verification rejects leakage.
