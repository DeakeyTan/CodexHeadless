# CodexHeadless 0.9.0-uat.1 Release Notes

Build date: 2026-07-11
Distribution: **Internal UAT Only - Unsigned**
Production status: not production-ready

## Scope

This controlled UAT build provides menu-bar and CLI workflows for Keep Awake, physical or managed-virtual display handoff, built-in display soft-disconnect, Touch Bar UI hiding, confirmation, rollback, Restore, and recovery diagnostics.

R7 adds exact helper-reported virtual-display ID binding, structured Clean Normal classifications, token/identity-based orphan-helper detection, confirmation-dialog Restore suppression, cross-process logger testing, and an activated XCTest suite.

## Tested Status

- Build host: Apple Silicon, macOS 27.0, Xcode 27.0 beta.
- Automated XCTest and universal artifact verification: see the R7 verification reports.
- Apple Silicon hardware workflow matrix: NOT VERIFIED until the manual report is completed.
- Intel binary is included, but Intel UAT validation is incomplete.
- Signing, notarization, and stapling: NOT STARTED.

## Safety Notice

CodexHeadless uses private macOS display APIs. Behavior can differ by Mac model and macOS release. Preserve local or SSH terminal access, learn the Restore hotkey, avoid critical unattended work, and Restore to verified Clean Normal before uninstalling.

## Known Limitations

- Unsigned builds may trigger Gatekeeper prompts.
- Private display and Touch Bar APIs can change across macOS updates.
- Hardware cycle, sleep/soak, and Intel matrices remain listed in `V0.9_R7_Manual_Stability_Report.md`.

Report results with `V0.9_R7_UAT_Feedback_Template.md` after removing unrelated sensitive log content.
