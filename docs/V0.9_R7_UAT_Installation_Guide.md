# CodexHeadless 0.9.0-uat.1 Installation Guide

Status: **Internal UAT Only - Unsigned**

## Prerequisites

- macOS 13 or newer for this package build.
- Apple Silicon is the primary UAT architecture. Intel binary inclusion does not imply completed Intel validation.
- Administrator access for PKG/CLI installation.
- Local or SSH terminal access for recovery.

## Install

1. Verify the supplied SHA-256 values against `checksums.txt`.
2. Move `CodexHeadless.app` to `/Applications`, or open the supplied unsigned PKG.
3. Do not disable Gatekeeper globally. For an expected unsigned-build warning, use Finder's **Open** action or System Settings' per-app approval.
4. Optional CLI: install the supplied universal `codex-headless` as `/usr/local/bin/codex-headless` with executable permissions.
5. Launch `/Applications/CodexHeadless.app` and grant Accessibility permission only when macOS requests it for configured fallback/hotkey behavior.

## First Verification

Run:

```bash
codex-headless version
codex-headless status
codex-headless doctor
```

Confirm version `0.9.0-uat.1`, `Mode: Normal`, and `Safety Classification: clean`. Start with a physical external display and retain terminal access.

## Safe First Test

1. Note Restore hotkey `Control+Option+Command+Shift+R`.
2. Select Enable, verify the replacement display, and Confirm if prompted.
3. Select Restore and wait for verified Normal.
4. If any result differs, follow `RECOVERY_GUIDE.md` and preserve the log.

## Uninstall

1. Run `codex-headless off` until status is Normal and clean with no active Journal.
2. Quit the App.
3. Run `scripts/uninstall.sh` from the source package, or remove `/Applications/CodexHeadless.app` and `/usr/local/bin/codex-headless` manually.
4. Never uninstall while Restore is paused, cleanup is incomplete, or Recovery Required is shown.
